# AI Marketing Tool

## Comprehensive Project Information
The AI Marketing Tool is a powerful application designed to help businesses automate their marketing processes using AI technologies. The tool is built with Python using FastAPI for the backend and React for the frontend, providing a seamless user experience.

## Features
- **Lead Generation**: Automatically collect and manage leads from various sources to boost your sales funnel.
- **Content Creation**: Generate marketing copy and content tailored to your target audience using AI algorithms.
- **Automation**: Automate repetitive marketing tasks, freeing up time for more strategic activities.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/zrohitJha/ai-marketing-tool.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd ai-marketing-tool
   ```
3. **Set up the backend**:
   - Navigate to the backend directory.
   - Install dependencies:
     ```bash
     pip install -r requirements.txt
     ```
   - Run the FastAPI server:
     ```bash
     uvicorn main:app --reload
     ```

4. **Set up the frontend**:
   - Navigate to the frontend directory.
   - Install dependencies:
     ```bash
     npm install
     ```
   - Start the React application:
     ```bash
     npm start
     ```

5. **Access the application**:
   Open your browser and go to `http://localhost:3000` to access the AI Marketing Tool.

## Conclusion
The AI Marketing Tool is a comprehensive solution for enhancing your marketing efforts through AI. With features geared towards lead generation, content creation, and automation, it's designed to streamline your marketing process and improve efficiency.
